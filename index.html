ğŸ§© Variabel & Tipe Data

Variabel adalah sebuah "kotak" atau "wadah" untuk menyimpan data dalam program. Setiap variabel memiliki nama dan menyimpan suatu nilai.

ğŸ¯ Apa Itu Variabel?

Bayangkan variabel seperti kotak yang memiliki label (nama). Kamu bisa menyimpan barang (nilai) di dalam kotak tersebut, dan kapan saja kamu bisa membuka kotak untuk mengambil atau mengubah isinya.

nama = "Bryan" # variabel bernama 'nama' berisi teks "Bryan"
umur = 16 # variabel bernama 'umur' berisi angka 16
tinggi = 171.5 # variabel bernama 'tinggi' berisi angka desimal 171.5
adalah_pelajar = True # variabel bernama 'adalah_pelajar' berisi True

ğŸ”§ Function (Fungsi)

Function adalah blok kode yang dapat digunakan kembali untuk melakukan tugas tertentu. Function membantu membuat kode lebih rapi dan mudah dirawat.

ğŸ¯ Mengapa Perlu Function?

  • âœ” Menghindari penulisan kode yang berulang
  • âœ” Membuat kode lebih terstruktur dan mudah dibaca
  • âœ” Memudahkan debugging dan maintenance
  • âœ” Meningkatkan reusability kode

ğŸ“ Struktur Function
